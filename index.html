About Me

AI Excellence with Purpose and Precision

I’m Hariom Suraj Kabra, a full-stack developer with a strong focus on backend development, particularly using Django, along with expertise in frontend technologies and database management. My journey in full-stack development has allowed me to create dynamic and scalable web applications that cater to real-world business needs.

I’ve worked on a range of projects that highlight my full-stack skills, including Resumecraft, a Django-based platform that allows users to generate personalized resumes. I built both the front-end and back-end, integrating Python, HTML, CSS, and JavaScript to ensure a seamless user experience. Additionally, I developed ApnaShop, an e-commerce platform where I designed both the user interface and the backend infrastructure, using Django for server-side development and integrating payment gateways for a complete shopping experience.

In these projects, I implemented robust database management systems, RESTful APIs using Django REST Framework (DRF), and ensured scalability and security across both the frontend and backend layers. I have hands-on experience working with relational databases like MySQL and PostgreSQL, and I’m proficient in creating API endpoints, handling data validation, and optimizing performance.

I am also comfortable with front-end technologies like React and which I’ve used to create responsive and user-friendly interfaces, complementing my backend work. Whether it’s developing dynamic web applications or ensuring that the back-end is optimized for speed and security, I’m passionate about building reliable, efficient systems.

With a deep understanding of both front-end and back-end development, I’m excited to continue developing scalable solutions that address complex challenges and improve user experiences
